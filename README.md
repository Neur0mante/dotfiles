# dotfiles

This repository contains my personal dotfiles for setting up a development environment.

## Scripts

- `bootstrap.sh`: Initial setup script to install tools and symlink dotfiles.
- `upgrade.sh`: Script to check and update installed tools safely.

## Usage

1. Clone the repository: `git clone https://github.com/Neur0mante/dotfiles.git ~/.dotfiles`
2. Run `./bootstrap.sh` to set up the environment.
3. Run `./upgrade.sh` periodically to update tools.
